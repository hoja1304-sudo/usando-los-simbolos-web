# Usando los simbolos

Replica web minimalista del juego de relaciones de orden para numeros de 100 a 999.

## Archivos

- `index.html`
- `styles.css`
- `app.js`
- `assets/`

## Uso

Abre `index.html` en un navegador. No necesita servidor ni dependencias externas.

El boton de audio activa o desactiva la lectura de consignas y retroalimentacion usando la voz del navegador.
